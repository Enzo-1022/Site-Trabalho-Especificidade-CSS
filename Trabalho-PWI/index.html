O que é a especificidade no CSS?

A especificidade no CSS, é o que define o quão especifico um seletor CSS é, portanto quanto mais específico, ele possui prioridade sob os outros.

Por tanto é por meio da especificidade que os navegadores definem quais valores e propriedades CSS serão aplicados há um elemento.


Hierarquia de Especificidade

Cada seletor CSS tem seu lugar na hierarquia de especificidade.

Existem quatro categorias que definem o nível de especificidade de um seletor:

  1. Estilos embutidos (inline) - Exemplo: <h1 style="color: pink;" >
  2. IDs – Exemplo: #navbar
  3. Classes, pseudoclasses, seletores de atributos - Exemplo: .test, :hover, [href]
  4. Elementos e pseudoelementos - Exemplo: h1, ::before

Como calcular a especificidade?

Comece em 0, adicione 100 para cada valor de ID, adicione 10 para cada valor de classe (ou pseudoclasse ou seletor de atributos), adicione 1 para cada seletor de elemento ou pseudoelemento, O estilo inline recebe um valor de especificidade de 1000 e sempre recebe a prioridade mais alta!, Há uma exceção a esta regra: se você usar a !important regra, ela substituirá até mesmo os estilos embutidos!

Seletor Valor Especificidade Calculo
p 1 1
p.teste 11 1+10
p#demonstracao 101 1+100
<p style="color: pink;"> 1000 1000
#demonstracao 100 100
.teste 10 10
.p.teste1.teste2 21 1+10+10
#navbar p#demonstracao 201 100+1+100
* 0 0

Seletores CSS

Os seletores CSS são usados para “encontrar” (ou selecionar) os elementos HTML que você deseja estilizar.

Podemos dividir os seletores CSS em cinco categorias:

  • Seletores simples (selecione elementos com base no nome, id, classe)
  • Seletores combinadores (selecionam elementos com base em um relacionamento específico entre eles)
  • Seletores de pseudoclasse (selecionam elementos com base em um determinado estado)
  • Seletores de pseudo-elementos (selecionam e estilizam uma parte de um elemento)
  • Seletores de atributos (selecionam elementos com base em um atributo ou valor de atributo)

Seletor de elementos CSS simples.

O seletor de elementos seleciona elementos HTML com base no nome (tag name) do elemento.


Seletor de id CSS.

O seletor de id usa o atributo id de um elemento HTML para selecionar um elemento específico.

O id de um elemento é único em uma página, então o seletor de id é usado para selecionar um elemento único!

Para selecionar um elemento com um id específico, escreva um caractere hash (#), seguido do id do elemento.

Lembrando que para declarar o id de um elemento não podemos utilizar números (apenas no início),
caracteres reservados do CSS/HTML (<, >, (), #, . , acentuações nas palavras etc...) também não pode conter espaços entre as palavras.


Seletor de classe CSS.

O seletor de classe seleciona elementos HTML com um atributo de classe específico.

Para selecionar elementos com uma classe específica, escreva um ponto final (.), seguido do nome da classe.

Quando declaramos uma class também não podemos utilizar números (apenas no início),
caracteres reservados do CSS/HTML (<, >, (), #, . , acentuações nas palavras etc...) (fonte pequena como se fosse um comentário)


Também podemos especificar que apenas alguns elementos HTML específicos devem ser afetados por uma classe, colocando o nome do elemento HTML antes de se referir a class.


Além disso podemos declarar mais de uma class para um elemento HTML.

Seletores combinadores CSS.

Um seletor CSS pode conter mais de um seletor simples. Entre os seletores simples podemos incluir um combinador.


Existem quatro combinadores diferentes em CSS:

  • seletor descendente (espaço)
  • seletor filho (>)
  • seletor de irmão adjacente (+)
  • seletor geral de irmãos (~)

Seletor Descendente

O seletor descendente corresponde a todos os elementos descendentes de um elemento especificado.


Seletor filho (>)

O seletor filho seleciona todos os elementos que são filhos de um elemento especificado.


Seletor de irmãos adjacentes (+)

O seletor irmão adjacente é usado para selecionar um elemento que está diretamente após outro elemento específico.

Os elementos irmãos devem ter o mesmo elemento pai e "adjacente" significa "imediatamente seguinte".


Seletor Geral de Irmãos (~)

O seletor geral de irmãos seleciona todos os elementos que são próximos irmãos de um elemento especificado.


Pseudo-classes CSS.

Uma pseudo-classe CSS é uma palavra-chave adicionada a seletores que especifica um estado especial do elemento selecionado.


Veja alguns exemplos:


:hover

A pseudo-classe hover identifica quando o cursor do mouse passa por cima de um elemento, e aplica as propriedades CSS que foram declaradas dentro do bloco da pseudo-classe hover {}.


:link

A pseudo-classe link permite que você selecione os links dentro de um elemento.,
Ela seleciona todos os links, até mesmo os que não foram visitados, incluindo os links já estilizados em outras classes ou ids ou em até mesmo em outras pseudo-classes, permitindo alterar o estilo desses links.


:visited

A pseudo-classe visited serve para aplicar um estilo aos links que já foram visitados pelo usuário. Por questões de privacidade, os estilos que podem ser modificados usando este seletor são muito limitados.


O que são pseudoelementos?

Um pseudoelemento CSS é usado para estilizar partes especificadas de um elemento.


Alguns exemplos exemplo:


::first-line

O ::first-line pseudoelemento é usado para adicionar um estilo especial à primeira linha de um texto.


::first-letter

O ::first-letter pseudoelemento é usado para adicionar um estilo especial à primeira letra de um texto.


::marker

O ::marker pseudoelemento seleciona os marcadores dos itens da lista.


Seletor de [atributo] CSS

O [attribute]seletor é usado para selecionar elementos com um atributo especificado.


Seletor CSS [atributo = "valor"]

O [attribute="value"] seletor é usado para selecionar elementos com um atributo e valor especificados.


Seletor CSS [atributo~="valor"]

O [attribute~="value"] seletor é usado para selecionar elementos com um valor de atributo contendo uma palavra especificada.
